# Advent of Code 2025
This repository contains my solutions to the Advent of Code 2025 challenges.

## Install Dependencies
With [Conda](https://docs.conda.io/en/latest/) installed, run:
```bash
conda env create -f env.yml
conda activate aoc2025
```

## Run Solutions
To run the solution for a specific day, use the following command, replacing `day_01` with the desired day folder:
```bash
python day_01/solution.py
```
